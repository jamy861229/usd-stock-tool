# 美股匯率試算器（USD FX Calculator）

這是一個 **純前端、免安裝、免註冊** 的美股匯率試算工具。  
協助使用者快速評估：  
**股票漲跌是否足以抵銷匯率變動帶來的影響。**

---

## 🔧 功能說明

### 支援兩種計算模式
- **輸入買入 / 賣出價格 → 計算實際漲幅**
- **輸入預期漲幅 % → 回推賣出價格**

### 匯率試算
- 自行輸入美元買入 / 賣出匯率
- 計算「股票打平」與「安全打平」門檻

### 投資判斷提示
- 🔴 紅色：股票漲幅不足，匯率可能吃掉獲利
- 🟡 黃色：剛好打平，風險偏高
- 🟢 綠色：高於安全門檻，相對安全

---

## 🚀 使用方式

1. 輸入匯率買入 / 賣出價格
2. 選擇計算模式
3. 輸入股票價格或預期漲幅
4. 點擊「試算」即可查看結果
