# Dividend Return Calculator

## Tool Name

- 中文：股息報酬試算器(個股)
- 英文：Dividend Return Calculator

---

## Tool Purpose

幫助使用者快速判斷：

> 這筆配息投資，扣掉常見成本後，現金流是否真的有感

重點：

- 每年實際可領股息
- 扣除成本後是否仍具吸引力
- 是否達到有感現金流
- 補充保費影響可先手動粗估

---

## Inputs

必填：

1. 投入金額（台幣）
2. 殖利率（%）

選填：

3. 投資標的價格
- 用途：估算持有股數

4. 手續費 / 成本（%）
- 預設：0.1425%
- 定義：常見台股買入手續費估算
- 可自行調整

5. 補充保費估算率（%）
- 預設：空白
- 未填視為 0
- 用途：手動粗估補充保費影響
- 不是正式健保扣費規則計算

---

## Formula

### 基本假設

- 採單利概念
- 殖利率固定不變
- 不考慮股價波動
- 不考慮再投入
- 不納入股息稅負
- 補充保費僅提供手動估算，不自動套用正式制度條件

### Calculation

- 年股息收入 = 投入金額 × 殖利率
- 手續費 / 成本 = 投入金額 × 成本%
- 補充保費估算 = 年股息收入 × 補充保費估算率
- 年實際股息收入 = 年股息收入 - 手續費 / 成本 - 補充保費估算
- 實際報酬率 = 年實際股息收入 / 投入金額
- 每月現金流 = 年實際股息收入 / 12
- 預估股數 = 投入金額 / 投資標的價格

## Status Signal

### Green

- condition: `實際報酬率 >= 4%`
- label: `可行`
- message: `股息收入穩定且具吸引力，適合作為現金流來源`

---

### Yellow

- condition: `2% <= 實際報酬率 < 4%`
- label: `普通`
- message: `報酬有限，還需要評估是否符合現金流目標`

---

### Red

- condition: `實際報酬率 < 2%`
- label: `偏低`
- message: `股息收入不具吸引力，可能不值得只為了配息投入`

---

## Result Interpretation

必須輸出結論句，例如：

- 每月平均現金流已達有感門檻，適合拿來評估配息策略
- 整體報酬仍具吸引力，但每月現金流距離理想狀態還有落差
- 成本壓力比補充保費估算更明顯，需要先確認這筆投入是否值得
- 補充保費估算與成本已明顯壓低報酬，可能無法達到預期的被動收入效果

---

## Explanation Copy

說明區需包含：

- 手續費 / 成本預設以 0.1425% 作為常見台股買入手續費估算
- 補充保費估算率為手動粗估欄位，未填視為 0
- 本工具不會依健保正式扣費規則自動判定，若要精算，仍應依單次配息金額與健保署規定判斷
- 本工具假設殖利率固定，未納入股息稅負，也未考慮股價波動與再投入效果
