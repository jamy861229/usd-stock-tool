freelifeofoctopus.com

Core:
→ Decision-support tools (幫使用者做選擇)

---

A. Decision Tools（核心主線）

定位：
→ 快速、單頁、直接給結論

特性：
- one page
- one purpose
- interpretation-first
- local-first（目前）

子分類：

A1. 投資決策工具（已啟動）
- 美股匯率試算器（已完成）
- 外幣定存 vs 匯差（已完成 / V1）
- 股息試算器（next）
-（未來）保本 / 報酬 / 手續費

A2. 生活決策工具（預留）
-（未來擴充）
- 不限投資，但符合「幫做選擇」

---

B. Event Discovery Product（長期主產品）

定位：
→ 收藏 + 探索「期間限定活動」
→ 使用者不是來找活動，是來「管理想去的活動」
→ 一個讓使用者「收集、整理、決定要不要去」的活動工具

核心：
- 收藏（最重要）
- 行事曆
- 地圖
- 搜尋

未來：
- 推薦
- 個人化

特性：
- 會逐步平台化（但現在還不是）

---

C. Traffic Tools（流量工具）

定位：
→ 短期流量 / SEO / 話題

特性：
- 可快速開發
- 可獨立存在
- 不一定長期維護

範例：
- WBC 計算器
- 節慶活動整理

---

D. Platform Layer（能力層 / 尚未成形）

定位：
→ 共用能力（未 fully build）

目前只允許：

- local save（工具）
- 基本分享（未來）

未來可能：

- 收藏系統（進階）
- 帳號系統（很後面）
- 跨裝置同步（很後面）

⚠️ 規則：
→ 不主動擴張
→ 只有「需求逼出來」才做
