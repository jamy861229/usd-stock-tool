# Break-even Return Calculator

## Tool Name

- 中文：損益兩平報酬率試算器
- 英文：Break-even Return Calculator

---

## Tool Purpose

幫助使用者在投入前快速判斷：

> 這筆投資至少要賺多少報酬率，才不會被成本吃掉

重點：

- 手續費會讓報酬門檻提高多少
- ETF 內扣費用會讓整體報酬率降低多少
- 是否需要把賣出成本與交易稅納入判斷
- 若有價格，換算每股或每單位至少要上漲多少

---

## Inputs

1. 投入金額（台幣）
   - type: `number`
   - example: `1000000`

2. 手續費率（%）
   - type: `number`
   - default: `0.1425`
   - meaning: 買入手續費率

3. ETF 費用率（%）
   - type: `number`
   - example: `0.3`
   - meaning: 年度內扣成本，用於估算對報酬率的拖累

4. 是否扣除補充保費（開關）
   - type: `boolean`
   - default: `false`
   - rate: `2.11%`
   - meaning: 是否把報酬被補充保費扣除的影響納入保本率

5. 是否包含賣出成本（開關）
   - type: `boolean`
   - default: `false`
   - meaning: 是否將賣出手續費與交易稅納入計算

6. 價格（每股 / 每單位）（選填）
   - type: `number`
   - example: `100`
   - meaning: 用於轉換為每股或每單位需要上漲的金額

---

## Formula

### 基本

- 買入手續費 = 投入金額 × 手續費率
- ETF 費用影響 = 投入金額 × ETF 費用率

### 若包含賣出成本（開啟）

- 賣出手續費 = 投入金額 × 手續費率
- 交易稅 = 投入金額 × 0.3%
- 總交易成本 = 買入手續費 + 賣出手續費 + 交易稅

### 若未開啟（預設）

- 總交易成本 = 買入手續費

### 保本報酬率

- 基礎保本率 = (總交易成本 + ETF 費用影響) / 投入金額

### 若包含補充保費（開啟）

- 調整後保本率 ≈ 基礎保本率 / (1 - 2.11%)

### 價格轉換（選填）

若有價格輸入：

- 每股需上漲金額 = 價格 × 保本報酬率

---

## Results

### 主結果

- 保本報酬率（%）
- 需要賺回金額
- 每股需上漲金額（若有價格）

### 成本拆解

- 買入手續費（-X%）
- 賣出成本（若開啟）
- ETF 費用影響（-X%）
- 補充保費影響（若開啟）
- 基礎成本率

### 額外說明

若未包含賣出成本：

- `目前僅估算買入成本`

若包含賣出成本：

- `已包含完整交易成本（含交易稅）`

---

## Status Signal

### Green

- condition: `保本報酬率 <= 2%`
- label: `低成本`
- message: `成本門檻低，較容易先賺回成本`

### Yellow

- condition: `2% < 保本報酬率 <= 5%`
- label: `中等`
- message: `成本有感，需要確認報酬空間是否足夠`

### Red

- condition: `保本報酬率 > 5%`
- label: `偏高`
- message: `成本偏高，小幅報酬可能先被成本吃掉`

---

## Result Interpretation

需依是否包含賣出成本給不同語意：

- 未含賣出：此為初步門檻，未包含完整交易成本
- 已含賣出：此為完整交易門檻，包含進出成本

結論需提醒使用者：

- 報酬低於保本率時，仍可能只是打平成本
- 成本門檻偏高時，小幅報酬可能不具吸引力
- 若輸入價格，可用每股需上漲金額理解實際價格門檻

---

## Explanation Copy

本工具預設僅計入買入成本，用於快速判斷報酬門檻
可選擇納入賣出成本（含交易稅）以進行更完整評估
ETF 費用率屬於內扣成本，這裡用於估算對整體報酬率的拖累，不代表額外收款
本工具為簡化模型，未納入完整稅制、買賣價差、市場波動與不同券商費率
