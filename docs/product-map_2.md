# B. Event Discovery Product（長期主產品）

## 定位

收藏 + 探索「期間限定活動」

---

## 核心概念

- 使用者不是來找活動  
- 是來「管理想去的活動」

---

## 核心功能

- 收藏（最重要）
- 行事曆
- 地圖
- 搜尋

---

## 未來方向

- 推薦
- 個人化
- 活動整理 / 分類

---

## 特性

- 會逐步平台化  
- 但現在仍維持簡單結構

---

# 🧠 Product Principles（核心原則）

1. 一個工具只解一個問題  
2. 結果一定要有結論（不是只有數字）  
3. 優先簡單，不追求完整  
4. 不預設做平台  
5. 用假設（What-if）取代精準資料  
6. UX 一致（Tool Pattern）

---

想法 → Spec → Codex → 實作 → 復盤 → 收斂 → 下一個工具
